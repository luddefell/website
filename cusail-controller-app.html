cusail controller app

Navigation subteam controller hardware and embedded tooling

Worked on embedded system development for CUSail's autonomous sailboat navigation stack, connecting controller hardware to the broader sensing and actuation system used by the navigation subteam.

The controller design centers on a Raspberry Pi Pico-based board with power regulation and switchbox interfacing, supporting reliable communication between controller inputs and the sailboat's embedded navigation hardware.

technical work

  • Designed controller electronics in KiCad around a Raspberry Pi Pico, USB-C power input, buck regulation, and switchbox connector routing.
  • Worked across embedded systems, PCB design, and navigation hardware integration.
  • Built toward more reliable control interfaces for autonomous sailboat testing and debugging.
